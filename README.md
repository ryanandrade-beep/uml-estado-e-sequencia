## Qual usar em cada situação?

**Use o Diagrama de Estado quando a pergunta for:**
> "Quais situações esse objeto pode estar? O que faz ele mudar de uma para outra?"

Exemplos: ciclo de vida de um pedido, estados de uma conta bancária, fases de um usuário no sistema.

**Use o Diagrama de Sequência quando a pergunta for:**
> "Quem chama quem e em que ordem para esse caso de uso funcionar?"

Exemplos: fluxo de login, processo de compra, abertura de conta.
